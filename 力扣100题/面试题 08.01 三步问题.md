### 属于“动态规划”的一种，但是由于此例中的值太大，所以只用python实现最普通的思路，很可能时间上会挂
### 代码块：因此同一种思路，我们用C来写  
```C
int waysToStep(int n){
    long p[4]={0,0,1,0};
    long * l=&p;
    for (int i=0;i<n;i++)
    {
        l[3]=(l[0]+l[1]+l[2])%1000000007;
        l[0]=l[1];
        l[1]=l[2];
        l[2]=l[3];
    }
    return l[3];
}
```
