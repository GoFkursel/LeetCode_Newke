## 3. 确认计算顺序  
### 表达式：$f(k)=max${$f(k-1),H_{k-1}+f(k-2)$}  

```python
class Solution(object):
    def rob(self, nums):
        """
        :type nums: List[int]
        :rtype: int
        """
        cur=0
        mon=0
        for i in nums:
            mon,cur=cur,max(cur,mon+i)
        return cur
```
